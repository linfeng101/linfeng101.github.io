Markdown基本语法

1.文本格式化

1.1 标题

使用#号标记，可以表示1-6级标题， 随#的个数递增，一级标题字号最大，六级标题字号最小。

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

效果如下：

注意：

  • 最后一个#字符与标题中间留一个空格
  • 标题应该置于行首，如果放入表格中可能无法正确解析

1.2 字体

星号与下划线都可以，单是斜体，双是粗体，三是粗斜体。
个人更喜欢用星号，更明了。

代码 效果
*斜体* 斜体
**粗体** 粗体
***粗斜体*** 粗斜体
`标红` 标红

快捷键：
加粗Ctrl+B
斜体Ctrl+I


1.3 区块引用

Markdown 中引用通过符号 > 来实现。> 符号后的空格，可有可无。
在引用的区块内，允许换行存在，换行并不会终止引用的区块。如果要结束引用，需要一行空白行，来结束引用的区块。
代码：

>这里是个引用
这里是个引用
这里是个引用

效果：

这里是个引用
这里是个引用
这里是个引用

嵌套引用：
代码：

> 这里是个引用
> > 嵌套引用
> 
> 这里是个引用

效果：

这里是个引用

嵌套引用

这里是个引用


1.4 换行

Markdown换行的方式有很多种:

  • 直接在一句话后敲两个空格
  • 两句话之间加一个空行
  • 如果你在编辑的时候，想让一行文字在显示的时候换行，就在中间加<br>

1.5 删除线

markdown中可以给文字添加删除线
代码：

~~添加删除线的文字~~

效果：

添加删除线的文字


1.6 下划线

下划线的使用和html中类似，在需要添加下划线的文字首尾添加<u>文本</u>
代码：

<u>这行文字已被添加下划线</u>

效果：

这行文字已被添加下划线


1.7 分割线

Markdown中给出了多种分割线的样式，我们可以使用分割线让文章结构更加的清晰。
分割线的使用，可以在一行中用三个-或*来建立一个分割线，但是注意：写分割线前，要空一行之后写，否则会导致前一行字体放大。
代码（四种表示方法都可以）：

分割段（下面空一行）

---
***
- - -
* * *

效果：

分割段（下面空一行）

2.2 区块代码

如果是在一个块内需要引用代码，则在需要引用的代码块的前一行和后一行使用三个反引号，同时在前一个反引号后写入代码的语言。
代码：

#include <stdio.h>

int main(void)
{
	return 0;
}
​```

	return 0;

3.列表

3.1 无序列表

无序列表，使用*、+、-，再加一个空格作为列表的标记
代码：

* 无序列表 1
+ 无序列表 2
- 无序列表 3

效果：

  • 无序列表 1
  • 无序列表 2
  • 无序列表 3

3.2 有序列表

使用数字接着一个英文句点加空格表示一个有序列表， 注意前面的数字对列表没有影响
代码：

1. 有序列表 1
2. 有序列表 2
3. 有序列表 3

效果：

  1. 有序列表 1
  2. 有序列表 2
  3. 有序列表 3

3.3 列表嵌套

列表可以嵌套，添加tab缩进表示层次
代码：

1. 母列表
    1. 子列表1
        - 粗体
        - 斜体
        - 粗体和斜体
    2. 子列表2
2. 母列表
3. 母列表

效果：

  1. 母列表
    1. 子列表1
      • 粗体
      • 斜体
      • 粗体和斜体
    2. 子列表2
  2. 母列表
  3. 母列表

4.链接

4.1 行内式链接

首先来看行内式，只要在方块括号后面紧接着圆括号并插入网址链接即可
代码：

[链接名称](链接地址)
样例：
[LinfengCS](https://kreiia.cn/)

效果：

LinfengCS


4.2 页内导航

我们同样可以使用markdown实现页内导航，步骤如下：

  1. 先定义一个锚记
    <div id="footer"></div>
  2. 然后设置页内链接
    [到底部](#footer)

5.表格和公式

5.1 表格格式

表格使用|来分割不同的单元格，使用-来分隔表头和其他行

  • :——将表头及单元格内容左对齐
  • ——:将表头及单元格内容右对齐
  • :—–:将表头及单元格内容居中
    注意：都是英文字符

代码样例：

| 项目        | 价格   |  数量  |
| --------   | -----:  | :----:  |
| 计算机     | \$1600 |   5     |
| 手机        |   \$12   |   12   |
| 管线        |    \$1    |  234  |

效果：

项目 价格 数量
计算机 $1600 5
手机 $12 12
管线 $1 234

注意，表头下面的虚线为了更好的分隔表头和表格内容，长度随意。

vscode插件推荐：Markdown Table

6.图片

Markdown中插入图片的使用方式是：
代码：

![图片描述，可写可不写，但是中括号要有](图片地址，本地链接或者URL地址。)

7.转义特殊符号

Markdown 可以利用反斜杠来实现转义， 支持以下这些符号前面加上反斜杠来帮助插入普通的符号：

\   反斜线
`   反引号
*   星号
_   底线
{}  花括号
[]  方括号
()  括弧
#   井字号
+   加号
-   减号
.   英文句点
!   感叹号

样例：

\\
\*
\_
\+
\.
等等

效果：

\
*
_
+
.


8.脚注

脚注是对文本的备注，我们时长在论文中看到脚注，在Markdown中的使用方法。（但目前Hexo不支持脚注语法）
代码样例：

使用 Markdown[^1]可以效率的书写文档, 直接转换成 HTML[^2], 你可以使用 Typora[^T] 编辑器进行书写。
[^1]:Markdown是一种纯文本标记语言
[^2]:HyperText Markup Language 超文本标记语言
[^T]:NEW WAY TO READ & WRITE MARKDOWN.

效果：
使用 Markdown[^1]可以效率的书写文档, 直接转换成 HTML[^2], 你可以使用 Typora[^T] 编辑器进行书写。
[^1]:Markdown是一种纯文本标记语言
[^2]:HyperText Markup Language 超文本标记语言
[^T]:NEW WAY TO READ & WRITE MARKDOWN.

注意：脚注自动被搬运到最后面，请到文章末尾查看，并且脚注后方的链接可以直接跳转回到加注的地方。
